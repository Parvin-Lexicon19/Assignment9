Bish-Bosh 2.0

This program gets the Loop Number as well as the First and Second division numbers from the user. Then it writes numbers from 1 to 'Loop Number' but it replaces First-Number and Second-Number divisable numbers with Bish and Bosh, respectively. For numbers divisable with both, it prints Bish-Bosh.
This program is written as an Assignment for C# Programming at Lexicon.
